# 投资策略 Playbook

> 核心投资策略和经验教训，Claude Code 执行投资任务时必须遵循。

---

## 投资哲学

**价值成长投资**：在合理估值范围内，寻找具备长期成长性的优质标的。

---

## 核心策略框架

### 1. 投资范围

| 维度 | 标准 |
|------|------|
| **市场** | 美股市场 |
| **板块** | 机器人产业链（工业机器人、AI芯片、机器视觉、医疗机器人、自动化设备等）|
| **市值偏好** | 中等及以上市值（避免小盘股的流动性风险）|

### 2. 选股标准

**同时满足**：
- ✅ **成长性**：具备清晰的增长逻辑（技术领先、市场份额提升、行业扩张等）
- ✅ **估值合理性**：PE、PB 等估值指标处于合理区间（不追高估值）
- ✅ **机器人概念**：主营业务或核心增长点与机器人产业链直接相关

**核心估值指标**（基于2025年市场实际）：

**AI芯片/计算平台**（市场已re-rate，估值中枢上移）：
- TTM PE：35-50（合理区间），可接受至55（对龙头）
- Forward PE：25-35（关注成长性与估值匹配）
- **预期PEG**：0.5-1.2（高增长必须匹配合理估值，>1.2需谨慎，基于NVDA/AMD真实数据0.5-0.9）
- **实际PEG**：验证历史增长质量（<1.5为优秀，>2.5为偏高）
- P/S：20-30（反映增长溢价）
- 关键：增长质量优于绝对估值

**工业自动化/机器人本体**（历史中枢30左右）：
- TTM PE：25-35（历史合理区间），当前市场溢价至40-55
- Forward PE：20-28
- **预期PEG**：1.4-2.4（稳健增长要求，基于TER真实数据1.4-1.5）
- **实际PEG**：验证历史增长质量（<2.5为合理，>3.5为偏高）
- P/B：3-6
- 周期性较强，注意避开估值过高时期（如PE>50）

**机器视觉/传感器**（快速成长）：
- TTM PE：25-35（略高于工业自动化）
- Forward PE：20-30
- **预期PEG**：1.0-2.5（快速成长要求适中估值，基于CGNX真实数据1.0-2.3，区间扩大以容纳波动）
- **实际PEG**：验证历史增长质量（<2.5为合理，>3.5为偏高）
- P/S：8-15
- 关注收入增长率和市场份额

**医疗机器人**（龙头享有护城河溢价）：
- TTM PE：55-75（龙头公司合理范围）
- Forward PE：40-60
- **预期PEG**：1.5-2.8（医疗科技合理区间，基于ISRG真实数据1.5-1.99，考虑2025年增长预期下调）
- **实际PEG**：验证历史增长是否支持估值（<3.5为可接受，>4.5为偏高）
- P/S：20-25（对早期公司更适用）
- PS估值更适用早期阶段公司

**通用成长指标**：
- ROE > 15%
- 收入增长率：
  - AI芯片/视觉：>30%（高成长）
  - 工业自动化：15-25%（稳健成长）
  - 医疗机器人：20-30%（持续成长）
- 52周价格位置：避免在历史高位追涨

**⚠️ 提醒**：以上为参考区间，不是绝对标准。优质公司可以适当放宽，但需有明确成长逻辑支撑。

### 3. 行业配置权重

机器人产业链内部细分领域权重参考（**仅供参考，非硬性限制**）：

```
├─ AI芯片/计算平台 - 核心驱动层（技术核心，可适当偏重）
├─ 工业自动化/机器人本体 - 应用层核心（确定性较高）
├─ 机器视觉/传感器 - 关键感知层（技术迭代快）
└─ 医疗机器人/特种机器人 - 新兴细分（弹性大，需单独评估）
```

**配置原则**：
- 避免过度集中于单一子行业，分散技术路线风险
- 核心领域（AI芯片、工业自动化）可配置更多仓位
- 新兴领域需单独评估技术验证进展，配置比例较低
- 根据市场环境和估值水平灵活调整，**不设硬性强求**

### 4. 仓位管理

| 规则 | 说明 |
|------|------|
| **持仓数量上限** | 10 只股票 |
| **单只仓位** | 根据总资金和确定性分配，避免过度集中 |
| **分批建仓** | 理想价位分批买入，降低平均成本 |
| **最大仓位限制** | 单只股票不超过总资产的 25-30%（激进策略） |

**确定性评估（简化版）**：
将确定性分为 **高/中/低** 三个等级，考虑以下因素：
- 成长性：增长逻辑是否清晰，市场空间是否足够大
- 估值合理性：当前估值 vs 历史区间 vs 同业
- 竞争优势：市场份额、技术壁垒、品牌影响力
- 市场趋势：技术形态是否支持上涨趋势

**仓位分配建议**：
- **高确定性**：25-30%（激进策略上限）
- **中确定性**：15-20%
- **低确定性**：5-10%
- **投机性**：不参与（小于5%的仓位无实际意义）

### 5. 止损止盈策略

#### 止损规则
- **技术止损**：
  - 跌破关键支撑位（如前低、整数关口）
  - 跌破 MA50/MA200 等重要均线
  - 从买入价下跌 8-10%（初始止损位）
  - 跌破近期盘整平台

- **基本面止损**：
  - 买入逻辑发生变化（业绩不及预期预期）
  - 行业趋势逆转或竞争格局恶化
  - 公司出现重大负面事件（财务造假、高管丑闻等）

#### 止盈规则
- **估值止盈**（区分行业）：
  - 达到目标估值上限（如 Forward PE > 40-50）
  - **AI芯片**：预期PEG > 1.2，实际PEG > 2.5
  - **工业自动化**：预期PEG > 2.4，实际PEG > 3.5
  - **机器视觉**：预期PEG > 2.5，实际PEG > 3.5
  - **医疗机器人**：预期PEG > 2.8，实际PEG > 4.5

- **技术止盈**：
  - 跌破上升趋势线或关键支撑位
  - 出现明显顶部形态（双顶、头肩顶等）
  - 连续大涨后放量滞涨

- **分批止盈**：
  - 上涨过程中逐步减仓，锁定利润
  - 剩余仓位以MA50/MA200作为动态止盈位

### 6. 风险控制

- **不追高**：避免在 52 周高位附近买入
- **不盲目抄底**：等待明确的反转信号
- **定期复盘**：评估持仓逻辑是否变化
- **及时止盈/止损**：不情感化持有
- **FOMO 应对机制**：对热门股票设定24-48小时冷静期，避免冲动买入
- **恐慌应对**：市场暴跌时检查买入逻辑是否改变，而非盲目跟随情绪

### 7. 现金管理策略

**基本原则**：
- 不对现金部分过度焦虑，不强迫所有现金必须投资
- 现金是对冲不确定性和等待机会的弹药
- 未来投资机会总是存在，不必恐惧错过

**现金比例管理**：
- **市场过热期**（整体估值偏高）：保留 30-40% 现金
- **正常市场期**：保留 10-20% 现金
- **恐慌/低估期**：逐步加仓至 5-10% 现金

**调仓触发条件**：
- 单只股票涨幅达到 25-30% 以上，评估是否需要调整仓位或止盈
- 总价值偏离配置权重超过 10%，考虑再平衡（避免频繁交易）
- 发现新的高确定性标的但现金不足时，按持仓质量决定卖出哪只
- 定期审查：每季度评估持仓质量和配置合理性，而非高频调整

**原则**：减少交易频率，避免过度交易。好股票应该长期持有，而非频繁买卖。

## 分析框架

执行投资分析时，遵循以下步骤：

```
1. 行业研究 → 机器人产业链细分领域机会
2. 个股筛选 → 按市值、成长性、估值筛选
3. 基本面分析 → 财务指标、竞争力、增长驱动
4. 估值判断 → 当前估值 vs 历史区间 vs 同业对比
5. 技术分析 → 52周位置、价格趋势
6. 买入策略 → 目标价位、分批计划、止损位
7. 仓位决策 → 根据确定性和总持仓情况分配
```

**估值对比详细指南**：

**历史估值区间**：
- 收集过去3-5年的PE、PB、PS数据
- 识别20/50/80分位数作为低估/合理/高估的参考
- 关注估值中枢的长期趋势

**同业估值对比**：
- 选择3-5家业务模式和规模相似的直接竞争对手
- 对比当前估值水平和历史估值差异
- 分析估值差异的合理性（竞争优势、成长性差别）

**增长调整的估值（PEG）**：
- **预期PEG（推荐用于成长股）**：Forward PE ÷ 未来3年预期增长率
  - 反映前瞻估值与预期增长的匹配度
  - 医疗机器人预期PEG合理区间：1.5-2.8（基于ISRG真实数据1.5-1.99，考虑2025年增长预期下调）
  - AI芯片/计算平台预期PEG合理区间：0.5-1.2（基于NVDA/AMD真实数据0.5-0.9，高增长要求）
  - 工业自动化预期PEG合理区间：1.4-2.4（基于TER真实数据1.4-1.5，稳健增长）
  - 机器视觉预期PEG合理区间：1.0-2.5（基于CGNX真实数据1.0-2.3，区间扩大以容纳波动）
  - 预期PEG > 3.0：明显高估，需谨慎

- **实际PEG（验证历史增长质量）**：TTM PE ÷ 过去3-5年历史增长率
  - 反映当前估值与历史实际增长的对比
  - 实际PEG > 4.0：历史增长质量支撑不足
  - 实际PEG < 2.0：历史增长质量优秀，估值有支撑

- **双PEG分析原则**：
  - 同时计算预期PEG和实际PEG，进行对比分析
  - 差异率 = (预期PEG - 实际PEG) ÷ 实际PEG
  - 差异率 > +30%：预期过度乐观，需验证增长加速的合理性
  - 差异率 < -30%：预期过度悲观，可能存在买入机会
  - 不采用混合PEG（TTM PE ÷ 预期增长率）

**机器人产业链内部对比**：
- 在相同子行业内对比估值水平
- 分析估值差异的原因（技术路线、市场份额、增长预期）
